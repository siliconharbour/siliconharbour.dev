# Add TechNL job board aggregator

https://technl.ca/job-seekers/ is a WP Job Manager board that aggregates jobs from CGI, Focus FS, NetBenefit, Telelink, etc. Provides a clean RSS feed at https://technl.ca/?feed=job_feed with structured fields (job_listing:company, job_listing:location, job_listing:job_type, job_listing:salary, full content in content:encoded).

Goal: a /manage/import/jobs/technl board view that lets me see all current TechNL listings (with company match status) and a sync button, plus matching MCP functions so my agent can poll and surface new jobs for me to manually createJob.

Plan:
1. Lib: app/lib/technl-jobs.server.ts — fetchTechNLJobs() that parses the RSS feed, returns { title, link, company, location, jobType, salary, descriptionHtml, descriptionText, pubDate }
2. Route: app/routes/manage/import/jobs.technl.tsx — table with company match (is company in DB? is job already imported via a direct source? is job already a manualJob with same URL?), sync button (no-op, the feed is always fresh on fetch)
3. Register in app/routes.ts
4. MCP bridge:
   - listTechNLJobs() — returns the same data the page shows, with company match info
   - getTechNLJob(link) — full description for one
5. Document new MCP functions in server.ts EXECUTE_DESCRIPTION
6. Add link from /manage/import/jobs to the TechNL board view
